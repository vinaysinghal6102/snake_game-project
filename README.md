# 🐍 Snake Game (Vanilla JavaScript)

A responsive and real-time Snake Game built using **Vanilla JavaScript**, supporting both **keyboard controls** and **mobile touch interactions**. This project demonstrates core JavaScript concepts, DOM manipulation, and game logic implementation.

---

## 🚀 Features

* 🎮 Real-time gameplay
* 📱 Fully responsive design (works on mobile & desktop)
* ⌨️ Keyboard controls for desktop users
* 👆 Touch controls for mobile devices
* 📈 Score tracking system
* ⚡ Smooth animations and performance
* 💥 Collision detection (wall)
* 🔄 Game restart functionality

---

## 🛠️ Tech Stack

* **JavaScript (Vanilla JS)**
* **HTML5**
* **CSS3**

---

## 🎯 How to Play

* Use **Arrow Keys** (↑ ↓ ← →) to control the snake on desktop
* Use **touch gestures / on-screen controls** on mobile
* Eat food to grow the snake and increase your score
* Avoid hitting walls or your own body
